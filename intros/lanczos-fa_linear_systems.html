Introduction

Positive definite systems

In the case that A\mathbf{A}A is positive definite, then the Lanczos-FA iterate is equivalent to the well-known conjugate gradient algorithm. The simplest proof of this amounts to showing the Lanczos-FA iterate satisfies the same optimality property as CG.

This optimality property allows us to derive a number of prior bounds for the convergence of Lanczos-FA (and equivalently CG).
