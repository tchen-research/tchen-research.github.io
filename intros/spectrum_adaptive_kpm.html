What’s new?

The Lanczos method for matrix function approximation (Lanczos-FA) can be used to approximate f(A)bf(\mathbf{A})\mathbf{b}f(A)b (read more about this task here). For polynomials, the method is mathematically exact, and so we can use this method to compute ⟨v∣pn(H)∣v⟩\langle \mathbf{v} | p_n(\mathbf{H}) | \mathbf{v} \rangle⟨v∣pn​(H)∣v⟩. In the present paper, the key observation is the expensive part of this approach (running Lanczos) is completely independent of the choice of reference density σ(x)\sigma(x)σ(x). This means we can obtain lots of different KPM approxiamtions for essentially free. This avoids the need for determining how to scale H\mathbf{H}H a priori and also allows us to use more “exotic” reference densities.

There is a lot of concern that the Lanczos method is unstable. Another contribution of this paper is to try and spread awareness about the way the instabilities of Lanczos actually impact methods like Lanczos-FA. For example, [1] shows that the method accurately computes (bounded) polynomial moments!
